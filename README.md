# Java Lessons

This repository tracks my journey to learn Java deeply, step by step, with focused practice and small runnable examples.

## Why This Repo Exists

- Build strong Java fundamentals, not just memorize syntax.
- Practice consistently with clear, small lessons.
- Keep code and notes in one place so progress is visible.

## Learning Goals

- Master core Java syntax and language features.
- Understand object-oriented programming deeply.
- Learn collections, exceptions, file handling, and generics.
- Practice problem-solving and clean coding style.
- Prepare for interviews and Java-focused exams.

## How To Run Current Lesson

From the repository root:

```bash
javac one/Main.java
java one.Main
```

Expected output:

```text
hello world
```

## Study Roadmap

1. Basics: variables, data types, operators, conditionals, loops
2. Methods and scope
3. OOP: classes, objects, constructors, inheritance, polymorphism
4. Packages and access modifiers
5. Exception handling
6. Collections and generics
7. File I/O and streams
8. Practice problems and mini-projects
